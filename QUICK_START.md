# 🚀 快速開始指南

## 第一步：準備圖示檔案

擴充功能需要圖示才能在 Chrome 中顯示。最簡單的方法：

### 使用線上工具（推薦）

1. 開啟瀏覽器，前往：https://www.favicon-generator.org/
2. 上傳任何圖片（可以是 Twitch logo 或 🎁 emoji 的截圖）
3. 下載生成的圖示包
4. 解壓縮後，找到這些檔案並放到 `icons/` 資料夾：
   - favicon-16x16.png → 重新命名為 icon16.png
   - favicon-32x32.png → 重新命名為 icon32.png
   - android-chrome-48x48.png → 重新命名為 icon48.png  
   - android-chrome-192x192.png → 重新命名為 icon128.png

### 或者使用現成的圖片

如果你有任何 PNG 圖片：
1. 使用小畫家或任何圖片編輯軟體
2. 調整大小為 128x128 像素
3. 另存 4 份，分別命名為：
   - icon16.png
   - icon32.png
   - icon48.png
   - icon128.png
4. 全部放到 `icons/` 資料夾

---

## 第二步：安裝擴充功能

### 在 Chrome 瀏覽器

1. 開啟 Chrome
2. 在網址列輸入：`chrome://extensions/` 並按 Enter
3. 在右上角開啟「開發人員模式」（Developer mode）
4. 點擊「載入未封裝項目」（Load unpacked）
5. 選擇這個專案資料夾：`e:\My Project\twtich_auto_click_btn`
6. 點擊「選擇資料夾」

### 在 Edge 瀏覽器

1. 開啟 Edge
2. 在網址列輸入：`edge://extensions/` 並按 Enter
3. 在左側開啟「開發人員模式」
4. 點擊「載入解壓縮的擴充功能」
5. 選擇專案資料夾：`e:\My Project\twtich_auto_click_btn`
6. 點擊「選擇資料夾」

### 確認安裝成功

- 你會在擴充功能列表中看到「Twitch Auto Claim Bonus」
- 瀏覽器工具列會出現擴充功能圖示

## 第三步：開始使用

### 1. 開啟 Twitch 實況

在瀏覽器中開啟任何 Twitch 實況主頁面，例如：
- https://www.twitch.tv/vedal987
- https://www.twitch.tv/shroud
- https://www.twitch.tv/any_streamer_name

### 2. 開啟控制面板

點擊瀏覽器工具列的擴充功能圖示，會彈出控制面板。

### 3. 檢查設定

- ✅ 確認「全域啟用」開關是開啟的（預設開啟）
- ✅ 在「開啟的實況」列表中，確認你的 Twitch 分頁有出現
- ✅ 每個實況主旁邊的開關預設都是開啟的

### 4. 自動運作

現在擴充功能已經開始運作了！

- 當 Claim Bonus 按鈕出現時，會自動點擊
- 控制面板會顯示統計資訊
- 領取成功時會顯示通知（可在設定中關閉）

## 常見使用場景

### 同時看多個實況

1. 開啟多個 Twitch 分頁（不同實況主）
2. 每個分頁都會自動領取
3. 在控制面板可以看到所有分頁
4. 可以單獨關閉某個實況主的自動領取

### 關閉特定實況主

1. 開啟控制面板
2. 找到要關閉的實況主
3. 點擊旁邊的開關（變成灰色表示關閉）
4. 該實況主就不會自動領取了

### 查看統計

控制面板會顯示：
- **今日已領取**：今天總共領取幾次（午夜自動重置）
- **總計領取**：所有時間累計次數
- **最近活動**：最近 10 次的領取記錄

### 調整設定

在控制面板底部的「設定」區：
- **顯示領取通知**：領取成功時是否顯示桌面通知
- **播放提示音效**：領取時是否播放音效
- **檢查間隔**：多久檢查一次按鈕（2-10 秒）

## 疑難排解

### ❌ 看不到擴充功能圖示

**解決方法：**
1. 確認擴充功能已安裝（到 `chrome://extensions/` 檢查）
2. 點擊瀏覽器工具列右側的「拼圖」圖示
3. 找到「Twitch Auto Claim Bonus」並點擊「釘選」圖示

### ❌ 沒有自動點擊按鈕

**檢查清單：**
1. ✅ 全域啟用開關是否開啟？
2. ✅ 該實況主的個別開關是否開啟？
3. ✅ 是否真的有 Claim Bonus 按鈕出現？
4. ✅ 按 F12 開啟開發者工具 → Console，查看是否有錯誤訊息

**進階除錯：**
1. 開啟開發者工具（F12）
2. 切換到 Console 分頁
3. 應該會看到類似這樣的訊息：
   ```
   [Twitch Auto Claim] 已載入並開始監聽
   [Twitch Auto Claim] 開始自動點擊
   ```
4. 如果看到錯誤訊息，請回報

### ❌ 控制面板顯示「沒有開啟的 Twitch 分頁」

**可能原因：**
1. 你開啟的不是實況主頁面（例如只是 twitch.tv 首頁）
2. 頁面還在載入中

**解決方法：**
1. 確認網址格式是：`https://www.twitch.tv/實況主名稱`
2. 等待頁面完全載入後，點擊「重新整理」按鈕
3. 如果還是沒有，重新整理 Twitch 分頁

### ❌ 統計數字不正確

**解決方法：**
- 點擊控制面板底部的「清除記錄」按鈕
- 統計會重置為 0

### ❌ 擴充功能停止運作

**解決方法：**
1. 到 `chrome://extensions/` 
2. 找到「Twitch Auto Claim Bonus」
3. 點擊「重新載入」按鈕（圓形箭頭圖示）
4. 重新整理所有 Twitch 分頁

## 進階技巧

### 開機自動啟動（Windows）

讓 Chrome 在開機時自動啟動，這樣擴充功能也會自動運作：

1. 按 `Win + R`
2. 輸入 `shell:startup` 並按 Enter
3. 在開啟的資料夾中，建立 Chrome 的捷徑
4. Chrome 捷徑位置通常在：
   ```
   C:\Program Files\Google\Chrome\Application\chrome.exe
   ```

### 讓 Chrome 在背景執行

即使關閉所有視窗，Chrome 仍在背景執行：

1. Chrome 設定 → 進階設定
2. 系統 → 啟用「當 Google Chrome 關閉時繼續執行背景應用程式」
3. 這樣關閉視窗後，擴充功能仍會繼續運作

### 備份設定和統計

擴充功能的資料儲存在 Chrome 本地：
1. 如果要備份，可以使用「匯出資料」功能（未來版本可能新增）
2. 目前可以截圖控制面板來記錄統計
